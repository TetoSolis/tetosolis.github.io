WattSense (IoT)

Node-RED MQTT Raspberry Pi Python

"Domotiser sa maison, c'est passer 3 heures à configurer une prise pour gagner 3 secondes à l'allumer."

Bienvenue sur WattSense. Un projet IoT complet pour surveiller et automatiser la consommation électrique. Parce que savoir que le grille-pain consomme trop, c'est la première étape avant de décider de manger du pain froid.

🔧 Le Matos (Les Jouets)

On ne fait pas de l'IoT sans un peu de hardware :

  • 🔌 Prises Shelly : Les agents doubles. Elles contrôlent l'énergie et la rapportent au QG.
  • ⚡ Prise Meross MSS310 : La rebelle. Il faut négocier avec son API et récupérer une clé d'authentification pour qu'elle daigne nous parler.
  • 🍓 Raspberry Pi : Le cerveau de l'opération. Il héberge tout ce petit monde sans (trop) chauffer.

🏗️ L'Usine à Gaz (Software)

Une stack logicielle robuste pour gérer le flux de données :

Node-RED 🚦

Programmation visuelle. On relie des boîtes avec des fils virtuels pour créer des scénarios complexes (ou des boucles infinies, ça arrive).

Mosquitto 🐭

Le broker MQTT. C'est le facteur ultra-rapide qui livre les messages entre les prises et le serveur. Il ne perd jamais un paquet.

InfluxDB & MySQL 📊

La mémoire. InfluxDB pour les séries temporelles (beaux graphiques) et MySQL pour la structure (SQL classique).

Python 🐍

Le script qui supervise tout, détecte les anomalies et ajuste les priorités énergétiques. Le vrai patron, c'est lui.

🧩 Comment ça marche ? (La Magie)

C'est un ballet bien orchestré :

  • 1. Collecte : Node-RED et Python harcèlent les prises pour avoir leurs données via MQTT.
  • 2. Décision : "Il fait beau ?" "L'électricité est chère ?" -> Le script décide d'allumer ou d'éteindre.
  • 3. Visualisation : Un Dashboard Node-RED pour voir en temps réel combien coûte votre douche chaude.

🚀 Installation (Si vous avez du temps)

Pour reproduire ce chef-d'œuvre sur votre Raspberry Pi :

  • Installez Node-RED : sudo npm install -g --unsafe-perm node-red
  • Installez Mosquitto : sudo apt install mosquitto
  • Configurez XAMPP (Apache/MySQL) et créez la BDD solaire.
  • Branchez tout, et priez pour que le Wi-Fi tienne le coup.
