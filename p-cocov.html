Co'Voit NFC

PHP MySQL Fullstack Écologie

"Moins cher que l'essence, plus convivial que le bus, et codé à la main (sans framework JS qui meurt en une semaine)."

Bienvenue sur Covoiturage Nord Franche-Comté. Ce projet a un objectif simple : faciliter le partage de trajets entre conducteurs et passagers dans la région. Parce que rouler seul dans une voiture de 5 places, c'est triste (et ça coûte cher).

✨ Ce que ça fait (Les Features)

Une plateforme complète pour gérer vos déplacements sans prise de tête :

  • 🔐 Douane Numérique : Inscription et authentification sécurisées. On ne laisse pas entrer n'importe qui (les robots sont priés de rester dehors).
  • 🚙 Mode Conducteur : Publication de trajets avec date, heure et nombre de places. Idéal pour rentabiliser votre trajet Belfort-Montbéliard.
  • 🔍 Mode Passager : Recherche avancée pour trouver un chauffeur qui ne conduit pas trop mal.
  • 💬 BlaBlaChat : Système de messagerie intégré pour demander si le conducteur accepte les chats, les sandwichs au thon ou la musique techno.
  • 📜 Mémoire d'éléphant : Historique complet des trajets passés et futurs (pour prouver que vous étiez bien à l'heure).

🛠️ Sous le capot (Stack Technique)

Pas de superflu, juste du solide qui tourne partout :

Frontend 🎨

CSS / SCSS : Parce que le style, c'est important. Une interface propre, responsive, et faite maison.

Backend ⚙️

PHP (Natif) : L'indémodable. Pas de magie noire, juste de la logique pure et dure pour gérer les données.

Mémoire 🗄️

MySQL : Une base de données relationnelle robuste pour stocker utilisateurs, trajets et messages sans rien perdre.

Environnement 🚀

XAMPP : Le grand classique pour faire tourner Apache et MySQL en local sans casser son OS.

🚀 Installation (Pour les courageux)

Envie de lancer votre propre concurrent à Uber sur votre machine locale ? Voici la recette :

  • 1. Cloner : Récupérez le code (`git clone ...`).
  • 2. XAMPP : Installez-le, lancez Apache & MySQL, et croisez les doigts pour que les ports ne soient pas déjà pris par Skype.
  • 3. Data : Importez `covoiturage.sql` via PHPMyAdmin.
  • 4. Déployer : Glissez tout dans `htdocs` et admirez le résultat sur `localhost`.
