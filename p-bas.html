Boîte à Chaussettes (SAE 302)

Java UDP Swing Sockets 🧦

"Une application de chat où les messages arrivent... ou pas. C'est ça, la magie de l'UDP."

Bienvenue sur "Boîte à Chaussettes". Pourquoi ce nom ? Parce que traduire Socket littéralement par Chaussette dans tout le code source nous semblait être une excellente idée à 3h du matin. C'est une application de messagerie Client/Serveur entièrement codée en Java et communiquant en UDP.

✨ Les Fonctionnalités (Ce qui marche)

Une messagerie minimaliste mais robuste (tant que le réseau l'est) :

  • 🔐 Authentification : Login et mot de passe. Parce que même dans une boîte à chaussettes, on trie le linge.
  • 💬 Chat Temps Réel : Discutez avec vos amis instantanément. Enfin, à la vitesse d'un datagramme UDP perdu dans le cosmos.
  • 👥 Gestion d'Amis : Système de demande et validation. On ne mélange pas les serviettes et les torchons.
  • 📬 Messages Hors-Ligne : Le serveur garde vos messages au chaud si vous n'êtes pas là.
  • 📜 Mémoire courte : Historique des 10 derniers messages stocké localement. Ce qui se passe avant part au lavage.

🏗️ Architecture & "Chaussette"

Le code est structuré de manière modulaire, avec une nomenclature qui fera hurler les puristes :

La Chaussette 🧦

La classe Chaussette.java. C'est notre interface de communication UDP. Elle habille vos données pour qu'elles n'aient pas froid sur le réseau.

Le Serveur 🖥️

Le chef d'orchestre (ou le tiroir principal). Il gère les Sessions, stocke les messages en attente et fait le tri.

Le Client 👤

Équipé d'une superbe interface Swing (le charme du rétro) pour envoyer des requêtes SEND, UPDATE ou NOTIFY.

Protocole 📡

Un protocole maison. Pas de JSON, pas de XML, juste des virgules et du texte brut. Efficace, léger, et lavable en machine.

🚀 Installation (Old School)

Pas de `npm install` ici. On compile à l'ancienne, comme des vrais :

  • 1. Compiler : javac -d out src/*.java (Ça chauffe le processeur).
  • 2. Serveur : java -cp out Server (À lancer en premier, sinon c'est le vide).
  • 3. Clients : java -cp out Client (Lancez-en plusieurs pour vous parler à vous-même).
